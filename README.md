1. Frontend
    Разработка ориентирована на создание интуитивно понятного интерфейса (UI) и продуманного пользовательского опыта (UX) на базе библиотеки ReactJs, фреймворка NextJs и сборщика проектов Vite. Функционал включает: адаптивную вёрстку (desktop/tablet/mobile), бургер-меню, карточки товаров/услуг с описанием, интерактивную карту и форму обратной связи. Особое внимание уделено accessibility (доступности): предусмотрены настройки размера шрифта, переключение между светлой, тёмной темами и режимом для слабовидящих, а также поиск по разделам, семантическая вёрстка, навигация с клавиатуры. Проект кроссбраузерен и оптимизирован для всех типов устройств.

2. Backend
    В качестве серверной платформы выбран FastAPI. Учитывая прогнозируемо невысокий уровень нагрузок, в качестве системы управления базами данных (СУБД) используется SQLite, что обеспечивает простоту развертывания и достаточную производительность в рамках поставленных задач.

    API разработано в соответствии с принципами RESTful архитектуры и спецификацией JSON:API. Это обеспечивает стандартизированный формат обмена данными, предсказуемую структуру запросов и ответов, а также удобство интеграции с клиентскими приложениями.

    Регистрация и аутентификация пользователей реализованы на основе JWT-токенов. Токены сохраняются на стороне клиента в httpOnly cookie и передаются в заголовках запросов для авторизации доступа к защищенным ресурсам.

    Обработка комментариев реализована следующим образом: после отправки комментарий сохраняется в БД со статусом "на модерации". Публикация происходит автоматически через 3 дня без участия администратора. Для пользователей доступна опция подписки на уведомления — в этом случае на указанную почту приходит письмо о выходе комментария. Фоновая отправка писем делегирована Celery для асинхронного выполнения и избежания блокировок основного потока.

    Язык разработки - Python

    В системе предусмотрено 2 роли:

    **Пользователь**
    Описание эндпоинтов API:
    - Зарегистрироваться на сайте (через подтверждение кода на почте, сервис - SMTP)
    - Верифицировать почту
    - Выйти из аккаунта
    - Подписаться на рассылку
    - Отписаться от рассылки
    - Оставить комментарий
    - Удалить комментарий
    - Изменить комментарий
    - Просматривать комментарии других пользователей (пагинацией)

    **Администратор**
    Для администратора реализован интерфейс управления пользователями и контентом: доступен список всех зарегистрированных пользователей, а также функционал модерации комментариев (подтверждение/отклонение). Предусмотрена фильтрация записей по дате и статусу комментариев.

    Описание эндпоинтов API:
    - Получение всех пользователей
    - Верификация комментариев
