# MediaFlow - production-ready шаблон

**MediaFlow** — шаблонное решение для типовых задач в бэкенд-разработке.  

Сервис предназначен для отправки комментариев на сайт и включает готовую архитектуру с регистрацией, аутентификацией и авторизацией, реализующую современные концепции бэкенд-сервера.
Всего 16 эндпоинтов.

## Управление проектом

```bash
docker compose up   # запуск
docker compose down # остановка

*Структура проекта*

1. Frontend
    Содержит единственный файл index.html с настроенными NGINX и Dockerfile.

2. Backend

    Язык разработки - Python

    **Технологии**
        - Docker;
        - NGINX;
        - jwt-токены;
        - s3;
        - ci (pytest, black, factories, github actions);
        - профайлер;
        - celery;
        - redis;
        - кеширование;
        - smtp;
        - postgresql;
        - rate limiting;
        - миграции;
        - хеширование;
        - другие вспомогательные инструменты.
    
    **Пользователь**
        Описание эндпоинтов:
        - Отправить комментарий на почту автору;
        - зарегистрироваться на сайте;
        - аутентифицироваться через письмо на почту;
        - подтвердить код из письма;
        - выйти из аккаунта;
        - получить данные своего профиля (реализовано кеширование);
        - оставить комментарий;
        - удалить комментарий (с картинкой);
        - получить все комментарии;
        - получить все верифицированные комментарии;
        - получить все неверифицированные комментарии;
        - разместить картинку в комментарии.

3. Дальнейшая поддержка и анализ
    Для анализа производительности API подключен профайлер в middleware.

Код из данного проекта можно легко масштабировать и изменять в любых проектах любой сложности.
