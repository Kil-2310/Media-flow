Поставщик сервера и хостинга - TimeWeb

1. Frontend
Будет реализован полноценный и понятный UI и UX: адаптивный дизайн, бургер меню, карточки с описанием, интерактивная карта, обратная связь, оптимизация под различные браузеры, блок с настройками размера текста и темой оформления (темная, светлая, для слабовидящих), поиск по разделам ...

Сайт разбит на 7 страниц:
- Главная
- Культура
- История
- Экология
- Экономика
- Туризм
- Регистрация

Технологии разработки 
* HTML5
* CSS
* SASS (SCSS)
* JavaScript
* SEO (внутренняя оптимизация)

2. Backend
Серверная часть будет реализована на FastAPI и использованием базы данных (далее - БД) SQLite по причине малых нагрузок.

Спецификация API: JSON-RPC, JSON-API

Механизм регистрации и входа в аккаунт будет происходить через с JWT-токены и localStorage

Пользователь будет иметь возможность:
- Зарегистрироваться на сайте (через подтверждение кода на почте сервис - SMTP)
- Выйти из аккаунта
- Подписаться на рассылку
- Отписаться от рассылки
- Оставить комментарий
- Удалить комментарий
- Просматривать комментарии других пользователей с пагинацией

После отправки комментарий сохраняется в базе данных и получает статус "на модерации". Публикация на сайте происходит автоматически через 3 дня. Пользователь может подписаться на уведомления, чтобы получить письмо на почту о выходе комментария. При этом рассылка происходит с использованием Celery

Технологии разработки 
* Python
* FastAPI
* SQLite
* Celery (Redis)
* Docker
* SQL

3. Дальнейшая поддержка
Для отслеживания посещаемости, поведения пользователей и технического состояния сайта будут подключены системы веб-аналитики: Яндекс.Метрика и Google Search Console. Также планируется регистрация проекта в Яндекс.Вебмастер для управления индексацией сайта в поисковых системах.
